## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/menezs/ProvaAuto.git
```

Entre no diretório do projeto

```bash
  cd ProvaAuto
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run start
```

Acesse o link

```bash
  http://localhost:5000/
```
